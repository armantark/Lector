# Product Context: Lector Bot

## Why This Project Exists
Many Christian communities use Discord for fellowship and discussion. Lector Bot brings the ancient practice of daily lectionary readings into modern chat platforms, making it easy for communities to engage with Scripture together.

## Problems It Solves

### For Communities
- **Accessibility**: Not everyone knows where to find their tradition's lectionary online
- **Consistency**: Automated delivery ensures readings arrive daily without manual effort
- **Engagement**: Shared readings in a community channel spark discussion

### For Individuals
- **Convenience**: Readings delivered where you already spend time
- **Discovery**: Explore lectionaries from different Christian traditions
- **Integration**: Bible Gateway links make it easy to read full passages

## How It Should Work

### User Journey: Guild Admin
1. Invite Lector Bot to server
2. Use `!subscribe armenian #scripture-channel` to set up subscription
3. Use `!time 8am` to set delivery time
4. Readings arrive automatically each day

### User Journey: Community Member
1. See daily reading embed in subscribed channel
2. Click Bible Gateway links to read full passages
3. Discuss readings with community

## User Experience Goals
- **Simple**: Minimal commands to get started
- **Informative**: Rich embeds with title, readings, and source links
- **Reliable**: Readings arrive on schedule without intervention
- **Respectful**: Clear attribution to source websites
